# 2_Rayleigh
#==========================================
ジャンル  ：3D横スクロールアクション
開発時期  ：2023年10月 1ヶ月間
開発人数  ：1人
開発環境  ：DirectX9、C++
仕様ツール：VS2019、Maya、メタセコイア、PhotoShop
担当箇所  ：プログラミングとモデルは全て自作しました

#アピールポイント=========
テクスチャアニメーションの管理を外部ファイルに書き出して
自分以外の人にも管理できるようにしました。

また、プレイヤーの操作感についてはプレイヤーにストレスを与えないように
様々な人にテストプレイをしてもらい、こだわりました。

#苦労した点=========
テストプレイをしてもらった上での調整は苦労しました。
特にプレイヤーのアクションごとのパラメーターの調整に関しては、上級者が突き詰められ、
初心者でもクリアはできるようにするように調整するのは苦労しました


#==========================================
# 3_ジャガンバイ
#==========================================
ジャンル  ：3Dパーティーシューティング
開発時期  ：2023年10月～2023月12月 2ヶ月間
開発人数  ：5人
開発環境  ：DirectX9、C++
仕様ツール：VS2019、Maya、メタセコイア、PhotoShop
担当箇所  ：プレイヤーや敵の挙動、武器やギミックのプログラミングを担当しました

#アピールポイント=========
プレイヤー側から武器に対する操作がシンプルなものにして拡張性を持たせることで
様々な武器を制作することができました。

また、さまざまなインゲームのパラメーターを外部に出すことで
制作の後半では他のメンバーにレベルデザインを任せることができました。

#苦労した点=========
チーム制作ということで、メンバー間でのモチベーションの差について悩むことがありました。
私はその状況でも、メンバーに対して仕様のすり合わせを行うことでお互いの仕様に対する深めるようにしました。
そうすることで、そのメンバーもゲームの完成形が見えるようになり、制作に積極的になれたと考えています。
